## 6. 예측 파이프라인 코드 흐름 요약

### 푸드 예측 공식 (곱셈 파이프라인)

```
adjusted_prediction = base_WMA(7일)
    × holiday_coef
    × weather_coef
    × food_temp_cross_coef   (기온×푸드 교차)
    × food_precip_cross_coef (강수×푸드 교차)
    × weekday_coef           (DB 동적, 0.80~1.25)
    × seasonal_coef
    × assoc_boost
    × trend_adjustment
```

### 안전재고 공식

```
safety_stock = daily_avg × safety_days × disuse_coefficient
  - safety_days: 유통기한 그룹별 (ultra_short=0.7, short=0.8, medium=1.0)
  - disuse_coef: max(0.65, 1.0 - disuse_rate × 1.2)
  - 캘리브레이터 오버라이드: safety_days, gap_coefficient 조정
```

### 최종 발주량

```
need_qty = adjusted_prediction + lead_time_demand + safety_stock
           - effective_stock - pending_qty

if 유통기한 ≤ 1일 AND 푸드: effective_stock = 0 (전일 재고 무시)
```
