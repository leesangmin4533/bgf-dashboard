# Plan: 비푸드 과자 카테고리 유통기한 7일 전 알림

## 1. 개요

### 목표
46513 호반점의 과자 카테고리(015 비스켓/쿠키, 016 스낵류) 상품 중
유통기한 만료 7일 전에 "나에게 보내기" 카카오 알림 발송.

### 대상
- 매장: 46513 (호반점) — 테스트 후 전 매장 확대
- 카테고리: 015 비스켓/쿠키, 016 스낵류
- 단톡방: 비활성 (테스트 완료 후 활성화)
- 나에게 보내기: 활성

### 현재 상태
- inventory_batches에 과자 active 배치 244건 존재
- expiry_date가 product_details 실제값 기반으로 보정 완료
- 가장 가까운 만료: 2026-04-18 (약 18일 후)
- get_batch_expiring_items(days_ahead=7)에 이미 과자도 포함됨

## 2. 현행 구조와 변경점

### 현행: generate_alert_message()에 비푸드 섹션으로 통합

```
generate_alert_message()
├── 푸드류 (시간 기반): get_alert_items()
└── 비푸드류 (배치 기반): get_batch_expiring_items(days_ahead=7)
    └── 과자도 여기에 포함 (015, 016)
```

현재 이 함수는 `send_kakao_alert()`에서 호출되며,
category="food_expiry"로 나에게 보내기에 포함됨.

### 변경 필요사항

1. **비푸드 알림 스케줄 추가**: 현재는 폐기 시간대(02/10/14/22시) 스케줄에만 연동.
   과자류는 시간대 폐기가 아니라 **일 단위** 만료이므로 별도 스케줄 필요.
   → 매일 07:00 수집 완료 후 1회 발송 (아침에 확인)

2. **메시지 포맷 분리**: 푸드 폐기 알림과 별도 메시지로 발송.
   "[호반점] 과자류 유통기한 알림" 형태.

3. **점포명 포함**: 모든 메시지에 `[점포명]` 접두사.

## 3. 설계

### 3.1 스케줄

```
매일 07:30 (메인 수집 07:00 완료 후)
  → 각 매장별 비푸드 유통기한 7일 이내 상품 조회
  → 과자 카테고리만 필터 (015, 016)
  → 카카오 나에게 보내기 (category="food_expiry")
  → 단톡방: 비활성 (추후 활성화)
```

### 3.2 메시지 포맷

```
[이천호반베르디움점] 과자 유통기한 알림 (04/11)

D-7 이내 만료 3건:
  크라운)산도딸기         만료 04/18  잔여 2개
  해태)후렌치파이사과      만료 04/19  잔여 1개
  오리온)통크            만료 04/20  잔여 3개

총 3건 6개 — 할인/소진 검토 필요
```

### 3.3 구현 위치

| 항목 | 위치 | 변경 |
|------|------|------|
| 과자 만료 조회 | ExpiryChecker.get_batch_expiring_items() | 이미 구현 (days_ahead=7) |
| 메시지 생성 | ExpiryChecker 새 메서드 | `generate_snack_expiry_message()` |
| 스케줄 등록 | run_scheduler.py | 07:30 스케줄 추가 |
| 알림 발송 | run_scheduler.py wrapper | send_message(category="food_expiry") |

### 3.4 필터 조건

```python
WHERE ib.status = 'active'
  AND ib.remaining_qty > 0
  AND p.mid_cd IN ('015', '016')
  AND ib.expiry_date <= date('now', '+7 days')
  AND ib.expiry_date > date('now')
  AND COALESCE(ib.expiration_days, 0) NOT IN (9999, 999)
  AND COALESCE(pd.orderable_status, '가능') != '불가'
```

### 3.5 확장 계획

| 단계 | 내용 | 시기 |
|------|------|------|
| 1단계 | 과자 (015, 016) 나에게 보내기 | 즉시 |
| 2단계 | 음료 (039~045), 면류 (032) 등 확대 | 테스트 후 |
| 3단계 | 단톡방 활성화 | 검증 후 |
| 4단계 | 전 매장 확대 | 안정화 후 |
