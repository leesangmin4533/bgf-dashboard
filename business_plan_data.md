## 2. 시스템 주요 기능

### 2-1. 핵심 기능 (구현 완료)

| # | 기능 | 설명 |
|---|------|------|
| 1 | **AI 수요예측** | WMA + ML 앙상블, 카테고리별 전략 패턴 (food/beer/tobacco/ramen 등 12종) |
| 2 | **자동발주** | BGF 넥사크로 사이트 직접 발주 (Direct API + Selenium 폴백) |
| 3 | **수요패턴 분류** | 4단계 (daily/frequent/intermittent/slow), Croston/TSB 간헐수요 모델 |
| 4 | **ML 이중모델** | 개별 모델 + 그룹 모델, data_confidence 블렌딩 |
| 5 | **날씨 연동** | 기온/강수량 예보 기반 수요 보정 (4단계 강수 세분화) |
| 6 | **행사 관리** | 1+1/2+1 프로모션 자동 감지 및 발주량 조정 |
| 7 | **폐기 분석** | 원인 분류 (과잉발주/수요하락/유통기한관리), 자동 보정 |
| 8 | **신제품 관리** | 14일 라이프사이클 모니터링, 유사상품 기반 초기 발주 |
| 9 | **CUT 대체발주** | 발주중지 상품의 대체 보충 (동일 중분류 내) |
| 10 | **웹 대시보드** | 실시간 발주현황, 예측분석, 폐기분석, 카테고리 드릴다운 |
| 11 | **카카오 알림** | 발주 완료/이상 감지 시 카카오톡 알림 |
| 12 | **디저트/음료 판정** | 자동 발주 유지/정지 판단 시스템 |
| 13 | **급여일 효과** | 급여일 자동 감지 + 카테고리별 부스트/감소 계수 |
| 14 | **재고 TTL** | 유통기한 기반 재고 수명 관리 |
| 15 | **DB 자동 동기화** | PythonAnywhere 클라우드 DB 자동 업로드 |

### 2-2. ML 예측 모델 상세

| 항목 | 상세 |
|------|------|
| 알고리즘 | LightGBM (Gradient Boosting) |
| 앙상블 | ML + 통계(WMA) 적응형 블렌딩 (MAE 기반 가중치 0.1~0.5) |
| 피처 수 | **47개** (기본26 + 입고패턴5 + 그룹컨텍스트4 + 시간대6 + 날씨3 + 기타3) |
| 카테고리 그룹 | food / alcohol / tobacco / perishable / general (5그룹) |
| 학습 주기 | 매일 증분학습(23:45, 90일) + 주간 전체학습(일 03:00) |
| 성능 보호 | MAE 20% 악화 시 자동 롤백 |
| 특수 모델 | Croston/TSB (간헐수요), CategoryDemandForecaster (중분류 총량) |

### 2-3. 발주 파이프라인 (7단계)

```
Phase 1: 데이터 수집 (실시간 재고/판매/날씨/행사)
Phase 2: 상품 정보 조회 (Direct API, 500개/배치)
Phase 3: AI 수요 예측 (WMA → 계수 보정 → ML 앙상블)
Phase 4: 발주량 결정 (안전재고 + ROP + 프로모션 + 카테고리 총량)
Phase 5: 발주 필터링 (CUT/정지/비발주일 체크)
Phase 6: BGF 사이트 발주 저장 (Direct API → Batch Grid → Selenium 3단 폴백)
Phase 7: 검증 및 알림 (발주 확인 + 카카오톡 알림)
```

## 8. 경쟁 우위 및 차별점

### 8-1. 기술적 차별점

| 차별점 | 설명 |
|--------|------|
| **넥사크로 자동화** | BGF 리테일의 비표준 넥사크로 플랫폼 직접 제어 (경쟁사 불가) |
| **Direct API 발견** | 넥사크로 내부 API를 역공학하여 직접 호출 (속도 30배 향상) |
| **4단계 수요 분류** | daily/frequent/intermittent/slow 패턴별 최적 알고리즘 자동 선택 |
| **실시간 외부요인** | 날씨(기온/강수), 급여일, 공휴일 자동 연동 |
| **3단 폴백** | Direct API → Batch Grid → Selenium (100% 발주 성공률) |
| **매장별 ML** | 매장 특성에 맞는 개별 학습 + 그룹 모델 블렌딩 |

### 8-2. 시장 규모 (추정)

| 항목 | 값 |
|------|-----|
| 전국 CU 매장 수 | 약 17,000개 |
