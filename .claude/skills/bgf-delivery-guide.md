# BGF 배송/도착 가이드

> 발주 → 배송 → 도착 → 판매 → 폐기 전체 흐름과 카테고리별 배송 규칙

---

## 배송 차수 (1차/2차)

| 구분 | 1차 배송 | 2차 배송 |
|------|---------|---------|
| 상품명 끝자리 | `1` (예: 도)한돈불백정식**1**) | `2` (예: 도)한돈불백정식**2**) |
| 도착 시간 | 당일 **20:00** (저녁) | 익일 **07:00** (아침) |
| 배송처 | 냉장1차, 냉동1차 | 냉장2차, 냉동2차 |

### 판별 방법
```python
# src/alert/delivery_utils.py
def get_delivery_type(item_nm: str) -> str:
    if item_nm.endswith('1'):
        return '1차'
    elif item_nm.endswith('2'):
        return '2차'
    return 'ambient'
```

## 카테고리별 배송 규칙

### 푸드류 (당일/익일 배송)

| mid_cd | 카테고리 | 유통기한 | 1차 도착→폐기 | 2차 도착→폐기 |
|--------|---------|---------|-------------|-------------|
| 001 | 도시락 | 1일 | 당일 20:00 → 모레 02:00 (30h) | 익일 07:00 → 익일 14:00 (31h) |
| 002 | 주먹밥 | 1일 | 당일 20:00 → 모레 02:00 (30h) | 익일 07:00 → 익일 14:00 (31h) |
| 003 | 김밥 | 1일 | 당일 20:00 → 모레 02:00 (30h) | 익일 07:00 → 익일 14:00 (31h) |
| 004 | 샌드위치 | 2~3일 | 당일 20:00 → +3일 22:00 (74h) | 익일 07:00 → +3일 10:00 (75h) |
| 005 | 햄버거 | 2~3일 | 당일 20:00 → +3일 22:00 (74h) | 익일 07:00 → +3일 10:00 (75h) |
| 012 | 빵 | 3일 | 상품별 유통기한 적용 (use_product_expiry) | 동일 |

### 일반 상품 (익일 배송)

| 구분 | 배송처 | 도착 시간 |
|------|--------|----------|
| 냉장 (유제품/음료 등) | 씨제이오산냉장 | 익일 07:00 |
| 냉동 (아이스크림 등) | 씨제이오산냉동 | 익일 07:00 |
| 상온 (과자/라면/잡화 등) | BGF로지스 | 익일 07:00 |

## 발주 시간대별 배송 일정

### 정상 (07:00 발주)
```
금요일 07:00 발주
  ├─ 1차 상품: 금요일 20:00 도착 (당일)
  └─ 2차/일반: 토요일 07:00 도착 (익일)
```

### 지연 (10:00 이후 발주)
```
금요일 11:00 발주 → 발주일 토요일(03/28)로 자동 보정
  ├─ 1차 상품: 토요일 20:00 도착 (당일)
  └─ 2차/일반: 일요일 07:00 도착 (익일)
```

### 발주 마감
- **10:00** — 당일 발주 마감 (이후 발주 시 익일 발주로 자동 전환)

## 예측 날짜 기준

```python
# src/prediction/improved_predictor.py
if target_date is None:
    target_date = datetime.now() + timedelta(days=1)  # 내일 기준 예측
```

| 실행 시각 | 예측 대상일 | 발주일 | 1차 도착 | 2차 도착 | 비고 |
|----------|-----------|--------|---------|---------|------|
| 07:00 | 내일 (토) | 오늘 (금) | 오늘 20:00 | 내일 07:00 | 정상 — 예측과 도착 일치 |
| 11:00+ | 내일 (토) | 내일 (토) | 내일 20:00 | 모레 07:00 | 예측(토) vs 실제판매(일) 1일 차이 |
